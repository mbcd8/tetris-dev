сначала делаешь:

sudo apt update                               // обновляешь менеджеж пакетов 'apt', и свои программы .
sudo apt upgrade                              // обновляешь саму ubuntu .

1. sudo apt-get install vim                   // установка редактора vim . 
2. sudo apt-get install tmux                  // установка мультиплексера tmux .
3. sudo spt-get install git                   // утилита github в терминале . (пока ты не умеешь через терминал грузить файлы)
4. sudo apt-get install libncurses-dev        // графика ncurses . ( эту графику мы использовали в нашем тетрисе )
5. sudo apt-get install arduino               // для программирования микроконтроллеров arduino . ( платки что у меня видел )
6. sudo apt-get install gtkterm               // программа сниффер для прослушки портов. ( для микроконтроллеров )

как устанавливать .deb программы:

заходишь в папку где находится скачанная программа
если она в архиве (tar, zip, rar ) - распаковываешь
потом в терминале вводишь следующее:

sudo dpkg -i название.deb

если что то пошло не так при установке и вышла ошибка
запусти в терминале следующую комманду:

sudo apt-get install -f
это должно исправить ошибки.
